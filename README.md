代码文件分类源代码src和web页面
源代码src包括配置文件jdbc.properties以及6个包；分别是
  pojo：JavaBean包
  test：测试包
  utils：各种工具包
  dao：与数据库交互的dao包
  service：对web层与dao层交互的包
  web：提供前端jsp页面与service层交互的包
  
web这里面包含如下部分
  jsp页面
  lib包存放各种jar包
  css包存放css样式
  js包存放js代码
  fonts包存放字体图标
  imgs包存放页面开发需要的照片
  upload包存放我们上传需要更新的照片
